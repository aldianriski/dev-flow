# Security Policy

## Contact

Email: aldian.mar@gmail.com

## Scope

Covers the dev-flow scaffold and skill library. Out of scope: third-party
dependencies (report upstream) and output files generated by running the skills.

## Reporting a Vulnerability

1. Email the contact address above with subject "dev-flow security disclosure".
2. Include: affected file(s), reproduction steps, and potential impact.
3. Acknowledgement within 72 hours.
4. Do not open a public GitHub issue for security vulnerabilities.
